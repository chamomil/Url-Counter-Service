# Url-Counter-Service

A microservice that counts click-throughs on links.

Language: Go,
DB: Postgres, pgx driver

## Requirements: 
- counter creation
- going to the URL that is associated with the counter
- yaml with db config
- db ddl

## Additional:
- OpenAPI specification
- regex constraints
- list of existing counters
- counter statistics
- docker
- auth Basic
- http request logger
